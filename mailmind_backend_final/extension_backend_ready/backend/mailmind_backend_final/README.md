# MailMind Backend (final)

## Overview
- Backend ready for Render deployment.
- Server uses your OpenAI key (set OPENAI_API_KEY in .env or Render env).
- MongoDB URL already configured.

## Quickstart (local)
1. Copy `.env` values into Render environment variables or keep `.env` locally.
2. `npm install`
3. `npm start`

## Important
- Default JWT secret is set in .env. Change it in production.
- OPENAI_API_KEY placeholder requires your key.
